# Mobile Tests (Phase 2)

This folder is reserved for mobile testing via Appium against [Sauce Labs Demo App](https://github.com/saucelabs/my-demo-app-rn).

**Status:** In development. The framework is mobile-ready (folder structure, fixtures, naming convention all support mobile expansion). Tests will be added in a Phase 2 commit series.

When mobile tests are added, this README will be replaced with full setup and execution instructions.

## Planned coverage

- `MOB-LOGIN-001` — login with valid credentials
- `MOB-LOGIN-002` — login with invalid credentials
- `MOB-CART-001` — add product to cart
- `MOB-CHECKOUT-001` — complete purchase flow

See `ALLURE_TEST_IDS.md` Section 2 for the full mobile test ID convention.
